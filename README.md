# AI Productivity Analytics Dashboard

## Project Overview
This project is an AI Productivity Analytics Dashboard developed using Power BI and Excel. The dashboard analyzes employee productivity, task completion, working efficiency, and department performance using interactive visualizations and KPIs.

## Tools Used
- Power BI
- Microsoft Excel
- Power Query
- DAX

## Features
- Employee productivity analysis
- Department-wise performance tracking
- Task completion monitoring
- Attendance and overtime analysis
- Interactive filters and slicers
- KPI dashboards

## KPIs Used
- Total Employees
- Productivity Score
- Tasks Completed
- Average Working Hours
- Attendance Percentage
- Department Efficiency

## Dashboard Insights
- Identified high-performing employees
- Analyzed productivity trends
- Compared department performance
- Improved workforce decision-making

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Data Transformation
4. Dashboard Creation
5. KPI Development
6. Data Visualization

## Conclusion
This dashboard helps organizations track productivity, monitor employee performance, and make data-driven business decisions using Power BI analytics.
